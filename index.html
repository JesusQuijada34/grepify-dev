grepify | El sitio web #1 para filtrar canciones de spotify

¿Qué es?

grepify es una herramienta minimalista que fusiona la terminal con el alma de la música 🎧. Diseñada para desarrolladores, amantes del código y entusiastas del ritmo, esta app permite buscar canciones directamente desde la línea de comandos usando la API pública de Spotify. Ya no necesitas interfaces gráficas: simplemente escribe grepify "nombre de la canción" y escucha previews musicales mientras sigues en tu flujo productivo 🐚⚡️. Además, grepify incluye una interfaz web animada con visuales sincronizados al ritmo, modo claro/oscuro, gradientes suaves, pulsos musicales y una experiencia envolvente. Perfecta para ambientes educativos, proyectos creativos o simplemente para darle ritmo a tu jornada de desarrollo.

Ejemplo en Python

import requests

url = "https://api.spotify.com/v1/search"
headers = {"Authorization": "Bearer TU_ACCESS_TOKEN"}
params = {"q": "nirvana", "type": "track"}

res = requests.get(url, headers=headers, params=params)
for item in res.json()["tracks"]["items"]:
    print(f'{item["name"]} - {item["artists"][0]["name"]}')
