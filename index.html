grepify

¿Qué es?

grepify es una herramienta minimalista que fusiona la terminal con el alma de la música 🎧. Diseñada para desarrolladores, amantes del código y entusiastas del ritmo, esta app permite buscar canciones directamente desde la línea de comandos o desde una web visual con efectos animados y beats sincronizados.

Ejemplo en Python

import requests

url = "https://api.spotify.com/v1/search"
headers = {"Authorization": "Bearer TU_ACCESS_TOKEN"}
params = {"q": "nirvana", "type": "track"}

res = requests.get(url, headers=headers, params=params)
for item in res.json()["tracks"]["items"]:
    print(f'{item["name"]} - {item["artists"][0]["name"]}')
