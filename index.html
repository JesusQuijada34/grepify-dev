grepify | El sitio web #1 para filtrar canciones de spotify

¿Qué es?

🎧 grepify – La terminal que escucha grepify es una aplicación minimalista y poderosa que convierte tu terminal en un buscador musical inteligente con la estética y precisión de una shell, y el alma sonora de Spotify. Construido para desarrolladores, entusiastas del CLI y amantes de la música por igual, grepify es la perfecta fusión entre código y ritmo. 🔍 ¿Qué hace grepify? Busca canciones directamente desde tu terminal usando la API pública de Spotify (https://developer.spotify.com/). Muestra resultados en tiempo real con nombre del track, artista, duración y preview. Permite reproducir fragmentos de las canciones sin salir del terminal (o desde la interfaz web visual). Ofrece una web complementaria interactiva, con reproductor visual, modo claro/oscuro, y estética neón sincronizada con el audio. 🖥️ ¿Por qué desde la terminal? Muchos desarrolladores prefieren interfaces rápidas y limpias. grepify está diseñado para quienes sienten que abrir una app gráfica para buscar música es como invocar a un demonio pesado del RAM 🐘. En cambio: grepify "daft punk" ...y listo: aparecen los resultados más relevantes, junto al botón para escuchar previews e integrar tu flujo musical al desarrollo sin fricciones. ⚙️ Características técnicas 🌐 Integración con Spotify mediante autenticación OAuth (Client Credentials) 🧠 Búsqueda profunda por artistas, tracks y álbumes ⏱️ Resultados rápidos, limpios, sin sobrecarga visual 🎵 Preview de pistas con reproducción integrada 💡 Interfaz web sincronizada con pulsos musicales (visualizador) 🎛️ Personalización de estilo (modo claro / modo hacker) 🐚 Integración con bash, zsh o fish como alias 🌟 Interfaz Web Además del terminal, grepify cuenta con una interfaz web opcional (creada con HTML/CSS/JS puro), que ofrece: Un buscador minimalista pero potente Un reproductor visual que late al ritmo real de la música 🎶 Un bloque de código Python con botón de copiar Efectos animados, gradientes dinámicos y estilo neón Tema adaptable claro/oscuro Módulo educativo y agradecimientos a la comunidad 🧠 Para qué sirve Para desarrolladores que quieren ambientar su sesión de código con buena música sin tocar el mouse Para aprender a integrar APIs REST desde la terminal Para explorar música nueva y aprovechar previews sin instalar apps pesadas Como herramienta educativa para enseñar autenticación, consumo de APIs y visualización en web 💚 Créditos y agradecimientos especiales Con alma comunitaria y muchas líneas de código con cariño, agradecemos la colaboración e inspiración de: @MikelCT · mentor y motor de ideas ⚙️ @GaboAI34_Bot · AI que ayudó a darle forma al ritmo 🤖 @Dallivin · artista y alma visual 🎨 Gracias por ponerle música al código.

Ejemplo en Python

import requests

url = "https://api.spotify.com/v1/search"
headers = {"Authorization": "Bearer TU_ACCESS_TOKEN"}
params = {"q": "nirvana", "type": "track"}

res = requests.get(url, headers=headers, params=params)
for item in res.json()["tracks"]["items"]:
    print(f'{item["name"]} - {item["artists"][0]["name"]}')
